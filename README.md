# Sky 闇の破片を確認するサイト
[サイトにアクセスする](https://jackylaughs.github.io/SkyShardTime/)
## 概要
現在の時刻、破片が落ちる時間とその場所を表示します。  
次の開催時間、もしくは開催終了までの時間をカウントダウンします。  
デイリーが更新される、または破片が落ちる時間に自動でリロードします。  
## お願い
webコーディング初心者のためレイアウトが崩れたり、場所や時間が誤って表示される可能性があります。  
サイトの情報は参考程度に留めてください。  
また、本サイトは以下の情報を参考にしています。
